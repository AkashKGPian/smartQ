# SmartQ

SmartQ is a QR-based real-time queue management system for hospitals and pharmacies.

## Problem
Hospitals and pharmacies rely on physical queues, leading to crowding, confusion, and poor patient experience.

## Solution
SmartQ allows patients to:
- Join ( doctor and dispensary ) queues via QR code
- Track live token status
- Upload prescriptions and consultation results

Staff can:
- Manage queues in real time
- Call, serve, or skip tokens
- Mark medicine availability

## Tech Stack
- Node.js + Express
- MongoDB (Mongoose)
- EJS + Bootstrap
- Socket.IO
- Cloudinary
